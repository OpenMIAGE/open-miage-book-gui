OpenMIAGE Book GUI
=====================

Description
=====================
OpenMIAGE Book GUI est l'implémentation graphique des API d'OpenMIAGE Book.

OpenMIAGE fourni pour l'instant:
  * un système d'authentification : OpenM-ID / OpenM-SSO
  * un annuaire : OpenM-Book

Viendrons s'y ajouter tout un ensemble d'autres services.

OpenMIAGE est globalement un projet open-sources découpé en deux projets open-sources :
  * OpenMIAGE API
  * OpenMIAGE GUI

Vous pouvez suivre l'avancé de nos développements sur [www.open-miage.org](http://www.open-miage.org)

Prerequis
=====================
 * PHP 5.2.x min
 * CURL extension.
 * GMP extension (or Bcmath, but GMP is mutch faster)

Dependencies
=====================
        launch ./install.dependencies.php before all to install dependencies (See ./lib/* for dependencies)
